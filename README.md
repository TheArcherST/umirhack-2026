# Madrigal Diagnostics

Кейс «Агент внутренней диагностики инфраструктуры».

[https://lvalue.linkoo.dev/](https://lvalue.linkoo.dev)

Платформа централизованного управления агентами: демон (Rust) устанавливается на целевые хосты, получает задачи от бекенда (FastAPI), выполняет проверки и отправляет результаты обратно. Бекенд собирает телеметрию, материализует её в метрики, рёбра графа связности и метаданные хостов. Веб-интерфейс (React + Electron) позволяет просматривать статусы агентов, историю задач, результаты и логи.

Frontend: `react/`
Backend: `python`
Agent: `rust/hack_agent/`

## Функционал
- Центральный компонент для управления агентами и задачами.
- Агент, который регистрируется в системе, получает задачи и выполняет проверки.
- Хранение результатов задач и журналов выполнения.
- Добавления нового агента.
- Проверки состояния сервера.

## Киллер фичи
- Отображение сети из агентов виде графов.
- Возможность выполнять произвольные команды по cron.
- Создание API ключа для рабочего окружения. (получение информаци об агентах управление)
  ими в зависимости от роли)

- `Make`
- `Docker` + `Docker-Compose`

## Как деплоить
1. Установить Docker Engine и make
2. Заполнить .env.example в .env
3. Заполнить python/.env.example в python/.env
4. Заполнить nginx/default.conf.example в nginx/default.conf
5. Прописать `make up`
6. Можно ещё прогнать тесты через `make test`

## Бинарники агентов

Чтобы скомпилировать агентов под все платформы и релизнуть
эти бинарники в приложении, напишите `make artifacts version=1.0.0`.
